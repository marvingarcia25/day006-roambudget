# day6_RoamBudget — Travel Budget Tracker

Plan a trip's budget by category, then log spending against it as you go.

An ASP.NET app for travel budgeting. Create a trip with a total budget split across categories — food, accommodation, transport, activities, shopping — add expenses as you spend, and watch what's left per category and overall.

## What it does

- Trips with a total budget and per-category budgets
- Add / delete expenses by category
- Budget overview: spent vs remaining, per category and overall, with percent used
- Ships with a seeded demo trip to explore

## Stack

- ASP.NET Core (minimal API, .NET 8)
- In-memory `TripService`
- Vanilla JS + CSS frontend

## Running it

```
dotnet run
```
